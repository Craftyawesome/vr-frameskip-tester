# vr-frameskip-tester

A clone of https://testufo.com/frameskipping for VR headsets.

Take long exposure (around 1/10-1/2 second) photos through each lens (or one lens if there is one physical screen). A gap in the sequence of lit squares is a dropped frame. A brightness difference indicates it was displayed for a different length of time.

`vr-frameskip-tester [grid_size] [display_size_percent]`

The default is 8 17.
